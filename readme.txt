Η κεντρικη αρχιτεκτονικη της εργασιας ειναι ενα restful API στο Spring framework με Spring boot, με SPA front end γραμμενο σε vue.js, η οποια ειναι αντιστοιχη της angularjs.
Το κομματι του back end ειναι διαμορφωμενο σε Maven, με το οποιο γινεται το build.
Για το build του front end κοματιού απαιτείται node js με τον Package manager npm. Με npm build θα κατέβουν όλα τα dependencies και με npm run start η εφαρμογή
Το IDE που χρησιμοποιησαμε ειναι το InteliJ IDEA και το WebStorm της JetBrains.
Το API της εφαρμογή φιλοξενηθηκε σε Tomcat server, και η διαχειριση της βάσης εγινε με τη χρηση MySQL Workbench 6.3.
O Φάκελος earthcord περιέχει το front end κομμάτι της εφαρμογής
Απο τα ζητουμενα υλοποιηθηκε ο,τι ειχε να κανει με την περιηγηση στο χωρο, την εγγραφη και τη συνδεση χρηστων, την καταχωρηση και το κλεισιμο δωματιων / κατοικιων.
Δεν υλοποιηθηκε η υπηρεσια ανταλλαγης μηνυματων και το bonus ζητουμενο.

Προσοχή: Για την σωστή λειτουργία της εφαρμογής πρέπει πρώτα να κατευθυνθείτε στην σελίδα https://localhost:8443 και να πατήσετε ότι αποδέχεστε την μη ασφαλή σύνδεση.
Η κανονική εφαρμογή βρίσκεται στο http://localhost:8090
Επειδή όλα τα δεομένα αποστέλονται στο API μέσω SSL δεν χρειάζεται το frontend να είναι και αυτό σε SSL
